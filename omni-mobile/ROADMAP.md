# Roadmap

Omni is an alpha research project. Near-term priorities are:

1. Reproducible community builds and self-hosting.
2. A documented threat model and stronger action confirmation controls.
3. Screenshot and accessibility-text redaction before provider requests.
4. Device compatibility reports for major Android manufacturers.
5. Unicode-safe text entry and better keyboard handling.
6. Unit and instrumentation coverage for action parsing and validation.
7. Optional local/on-device model support.
8. Exploration of Android AppFunctions and OEM Computer Control APIs.

The project will not add features intended to bypass `FLAG_SECURE`, authentication challenges, Android privacy controls, or app-specific security protections.
